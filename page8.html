Pseudo elements trong CSS

KHÁI NIỆM

Pseudo-elements trong CSS là các từ khóa đặc biệt bắt đầu bằng hai dấu hai chấm ::

  • ::before

    • chủ động thêm thuộc tính ( display:block; ) và bắt buộc phải cho (content:"";) vào

    mỗi element thì chỉ được tồn tại 1 ::before hoặc after.

  • ::after
  • ::first-letter

    • Có thể css cho chữ cái đầu to lên

  • ::first-line
  • ::selection

    • bôi chuột vào sẽ có highlight
